
# Employee Attendance & Payroll Management System (EAPMS)



##  Features
- User Authentication (Login system)
- Clock In / Clock Out functionality
- Attendance tracking and history display
- Dashboard with summary metrics
- Payslip generation and viewing
- Basic validation and error handling

---

##  Technology Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database (Conceptual):** MongoDB  

---

##  System Components

### Functional Components
- Authentication Component  
- Attendance Management Component  
- Payroll Processing Component  
- Payslip Generation Component  
- Reporting & Dashboard Component  

### Middleware Components
- API Layer  
- Business Logic Layer  
- Database Access Layer  
- Authentication Middleware  

### UI Components
- Login Screen  
- Dashboard Screen  
- Attendance Screen  
- Payslip Screen
